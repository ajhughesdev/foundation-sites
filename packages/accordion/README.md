# @foundation/accordion (Foundation 7)

Accordion is the Foundation 7 accordion plugin.

## Usage (draft)

```js
import { createFoundation } from '@foundation/core';
import { accordion } from '@foundation/accordion';
import '@foundation/css/foundation.css';

const app = createFoundation({ plugins: [accordion()] });
app.init(document);
```

## CSS

- Full bundle: `@foundation/css/foundation.css`
- Component-only: `@foundation/css/components/accordion.css`
