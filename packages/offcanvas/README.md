# @foundation/offcanvas (Foundation 7)

OffCanvas is the Foundation 7 off-canvas drawer plugin.

## Usage (draft)

```js
import { createFoundation } from '@foundation/core';
import { offcanvas } from '@foundation/offcanvas';
import '@foundation/css/foundation.css';

const app = createFoundation({ plugins: [offcanvas()] });
app.init(document);
```

## CSS

- Full bundle: `@foundation/css/foundation.css`
- Component-only: `@foundation/css/components/offcanvas.css`
