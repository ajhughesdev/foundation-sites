# @foundation/tabs (Foundation 7)

Tabs is the Foundation 7 tabs plugin.

## Usage (draft)

```js
import { createFoundation } from '@foundation/core';
import { tabs } from '@foundation/tabs';
import '@foundation/css/foundation.css';

const app = createFoundation({ plugins: [tabs()] });
app.init(document);
```

## CSS

- Full bundle: `@foundation/css/foundation.css`
- Component-only: `@foundation/css/components/tabs.css`
